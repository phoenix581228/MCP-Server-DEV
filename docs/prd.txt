# 花社大無人機影片 - AI剪輯分鏡表生成POC專案

## 專案概述

開發一個AI驅動的系統，能夠基於新聞文章內容和影片場記分析數據，自動生成剪輯分鏡表。此POC專案將驗證從文本語義到影片片段的自動匹配與序列生成技術可行性。

## 核心目標

1. **數據工程突破**：解決JSON字串解析問題，建立結構化場記數據
2. **語義匹配引擎**：開發文本與影片內容的智能匹配系統
3. **分鏡表生成**：創建自動化的剪輯序列規劃系統
4. **系統整合驗證**：與BigDipper架構完整整合

## 技術架構

### 核心組件
1. **數據解析器**：JSON→結構化數據
2. **語義引擎**：sentence-transformers + 自定義匹配
3. **序列生成器**：LLM + 剪輯邏輯
4. **整合接口**：MCP + BigDipper

### 技術棧
- **AI模型**：Gemini 2.5 Pro, Claude, sentence-transformers
- **數據處理**：Python, pandas, numpy
- **向量處理**：FAISS, scikit-learn
- **輸出格式**：JSON, CSV, PDF, EDL
