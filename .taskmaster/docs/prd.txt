PROJECT: MCP Server 六邊形智能開發工具綜合性能評估與對比分析

OBJECTIVE:
對六個已整合的 MCP Server 進行全面性能評估，並與原生 Claude Code 進行能力對比，評估六邊形智能開發工具的整體效能與價值。

SCOPE:
1. 評估六個 MCP Server 的個別性能表現
2. 測試各 MCP Server 的協同工作能力
3. 與原生 Claude Code 進行功能對比
4. 分析六邊形架構的優勢與限制
5. 提供優化建議與未來發展方向

MCP SERVERS TO EVALUATE:
1. Zen MCP - 多模型協作與深度分析工具
2. Serena MCP - 專業編碼助手與項目管理
3. TaskMaster MCP - 任務規劃與項目管理
4. Context7 MCP - 技術文檔與知識庫查詢
5. Perplexity MCP - 網路搜索與即時資訊
6. OpenMemory MCP - 長期記憶與知識管理

EVALUATION CRITERIA:
1. 功能完整性 - 各工具是否涵蓋其設計目標
2. 響應速度 - 各工具的執行效率
3. 準確性 - 輸出結果的正確性與可靠性
4. 協同效果 - 多工具協作的流暢度
5. 使用體驗 - 工具易用性與學習曲線
6. 資源消耗 - CPU、記憶體、網路使用情況
7. 穩定性 - 錯誤率與恢復能力

COMPARISON WITH NATIVE CLAUDE CODE:
1. 功能覆蓋範圍對比
2. 執行效率比較
3. 程式碼品質分析
4. 開發速度評估
5. 錯誤處理能力
6. 可擴展性分析

DELIVERABLES:
1. 各 MCP Server 個別性能評估報告
2. 六邊形架構協同工作分析報告
3. 與原生 Claude Code 詳細對比表格
4. 性能測試數據與視覺化圖表
5. 最佳實踐指南與使用建議
6. 優化建議與未來發展藍圖

KEY REQUIREMENTS:
- 使用標準化測試案例確保公平比較
- 記錄所有測試過程與結果
- 提供量化數據支持結論
- 考慮不同使用場景的需求
- 評估整合成本與效益

SUCCESS METRICS:
- 完成所有六個 MCP Server 的評估
- 提供至少 10 個測試案例的比較數據
- 識別至少 5 個協同工作的優勢案例
- 提出至少 3 個具體的優化建議
- 產出一份完整的評估報告
